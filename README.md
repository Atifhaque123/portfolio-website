# Portfolio Website

This is my personal portfolio website built using Express.js, EJS, Bootstrap 5, and Font Awesome.

## Features
- Home page with introduction
- About page describing my background and interests
- Projects page displaying my work
- Contact page with a simple form
- Clean and responsive design

## Technologies Used
- Node.js & Express.js
- EJS templating
- Bootstrap 5
- Font Awesome
- Morgan (request logging)
- Cookie-parser

## How to Run
1. Install dependencies:
2. Start the server:
   npm start
3. Open your browser and go to:
   http://localhost:3000

## Folder structure
/public → CSS, images, scripts
/views → EJS templates
/routes → Route files
app.js → Main server file

## Deployment
- Deployed to **Render** as a Web Service.
- Branch: `main`, Region: Oregon (US West)
- Build: `npm install`  |  Start: `npm start`

Live URL: https://portfolio-website-vbgv.onrender.com/
